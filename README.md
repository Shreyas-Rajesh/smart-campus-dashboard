# ✨ About The Project

Smart Campus Study Dashboard is a comprehensive web application built to streamline academic life for both students and administrators. It replaces fragmented communication channels with a single, unified interface.

The platform offers a clean, section-based experience where users can:

- 📊 View personalized academic stats
- 📚 Access course syllabi and topics
- 🗓️ Check weekly timetables dynamically
- 📝 Track assignments and their status
- 📢 Stay updated with section-specific notices
- 🎯 Explore upcoming campus events

# ⚡ Getting Started

```bash
git clone https://github.com/Shreyas-Rajesh/groupt_project_temp.git
```

## 2️⃣ Install Dependencies

```bash
pip install flask
```

## 3️⃣ Run The Application

```bash

```

Open `http://127.0.0.1:5000` in your browser.
