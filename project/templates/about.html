About Smart Campus

Project Description

Smart Campus Study Dashboard is a web-based academic management platform designed to centralise all academic information for college students. It provides a unified interface for managing courses, timetables, assignments, notices, and events.

Objective

To create a beginner-friendly, fully functional web application using Flask and Bootstrap that simulates a smart academic dashboard for students and administrators, with all data stored in simple CSV and text files.

Features
  • Student and Admin role-based login
  • Section-based personalised dashboard
  • Course syllabus viewer with interactive panel
  • Weekly timetable rendered from JSON data
  • Assignment tracker with status badges
  • Digital notice board per section
  • Campus events listing with date strip
  • Feedback and contact form with CSV storage
Technologies Used
Python 3 Flask HTML5 CSS3 JavaScript (ES5) Bootstrap 5 CSV Storage JSON
